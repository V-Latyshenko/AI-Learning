Roadmap · Claude Opus 4.7

AI Operating System
for Video Production

Event-driven AI operating system for an outsourced video production shop: 4 editors + owner, one large client. Normalized data in Google Sheets as the operational ledger, Frame.io webhooks as the primary event source, Apps Script and webhook handler as the glue, Claude Code (Opus 4.7) as the dev and ops environment.

Phase v1 · 3 weeks
Control Tower
All events in Sheets, KPIs by formula, AI narrative every morning.
Phase v2 · 3 more weeks
Smart Layer
Router, Prompt Rewriter, Suggester, Cost analytics, Telemetry.
Phase v3 · opt-in
AI Acceleration
Ad Performance Analyst, Evaluator, RAG, Slack, Coordinator.

Event lifecycle — from Frame.io to AI narrative

Source
Frame.io / Drive / Ads API push event
2
Ingress
Webhook ack 200 OK · HMAC verify · idempotency check
3
Queue + Audit
Append raw event · enqueue for async processing
4
Enrich
Apps Script worker · API call · normalize · upsert to Sheets
5
Compute
KPI formulas · cost rollups · deterministic, no LLM
6
AI Narrate
Prompt / Skill reads ready numbers · writes insight
When to use what in Claude Code: Prompt vs Skill vs Subagent
📝
Prompt
for single contracts
One prompt file, one input-output contract. No supporting files or scripts. Good when the task is stable and repeats in one context.
prompts/kpi_analyst.md
prompts/router.md
🧰
Skill
for playbooks
Reusable workflow with metadata, references, scripts, progressive disclosure. Good for multiple modes (daily/weekly/drill-down) with supporting docs.
.claude/skills/kpi-analysis/
.claude/skills/ad-performance-analysis/
🤝
Subagent
for delegation
Specialized role with its own constraints and whitelist of tools/skills. Good for multi-step queries that coordinate several skills.
.claude/agents/coordinator.md
What each phase delivers to the business owner
Phase v1
Control Tower
Weeks 1–3 · required base
  • All Frame.io events auto-land in Sheets
  • KPIs computed by formulas in real time
  • AI narrative every morning at 08:00
  • Approval gate for state changes
  • Append-only audit log for trust
Phase v2
Smart Layer
Weeks 4–6 · advisor
  • Docs briefs normalize automatically
  • Router dispatches tasks to the right agent
  • Suggester advises who to assign a video to
  • Cost workbook + ROI tracking
  • Per-agent telemetry: tokens, latency, cost
Phase v3
AI Acceleration
Opt-in · advantage
  • Ad Performance Analyst — flagship AI case
  • Evaluator controls AI output quality
  • RAG for SOPs and brand rules
  • Slack alerts for anomalies
  • Coordinator subagent for multi-step
